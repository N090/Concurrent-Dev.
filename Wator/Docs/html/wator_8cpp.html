move ( )

Checks that the Cell at the x, y co-ordinate is a movable fish or shark (that hasn't moved this game loop) and calls into their respective move functions.

Thread-safe move function that loops over the entire grid (split by the current number of threads) and calls into the single-cell move function per cell.

Parameters
x- The x co-ordinate of the cell
y- The y co-ordinate of the cell
